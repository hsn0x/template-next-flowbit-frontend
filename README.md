## Run Locally

Clone the project

```bash
  git clone https://github.com/hsn0x/indeed-express-sequelize-backend.git
```

Go to the project directory

```bash
  cd indeed-express-sequelize-backend
```

Install dependencies

```bash
  npm install
```

Start the server on develpment

```bash
  npm run dev
```

Start the server

```bash
  npm run start
```
